# Skystore

Интернет-магазин цифровых продуктов: рассылки, Telegram-боты, утилиты, веб-приложения.

## Технологии
- Python 3.10+
- Django
- Bootstrap 5

## Установка
1. Клонируйте репозиторий
2. Установите Poetry: https://python-poetry.org/docs/
3. Установите зависимости:
   ```bash
   poetry install
